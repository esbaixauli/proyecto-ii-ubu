See and modify users

An administrator can change a user's password, as well as associating and deassociating cases. Remember that if a user is deassociated from a case, its use stats will be eliminated

Create users

To create a user, simply insert his username and password. The field 'name' can't be repeated or empty nor can it contain spaces or special characters. You can also associate cases to that user through the checkboxes each case has, available in the lower panel.
