Ver/crear usuarios

Ver y modificar usuarios

Un administrador puede modificar la contraseña de un usuario, así como asignarle y desvincularle casos. Recuerde que si desvincula a un usuario de un caso, sus estadísticas de uso serán eliminadas

Crear usuarios

Para crear un usuario, simplemente introduzca su nombre de usuario y su contraseña. El campo nombre no puede estar repetido o vacío ni contener espacios o caracteres especiales. También puede asignarse casos a dicho usuario mediante las casillas de verificación de cada caso, disponibles en el panel inferior.
