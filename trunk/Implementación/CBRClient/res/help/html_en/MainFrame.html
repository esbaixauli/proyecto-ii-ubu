Administrator actions



Gestionar Tipos de caso

Un tipo de caso es un conjunto Problema-Solución del que se almacenarán casos concretos. El botón de Gestionar Tipos permite definir, modificar y borrar dichos Tipos de Caso, estableciendo de qué atributos se compone, como se evaluan por defecto y como se permite realizar una consulta en dicho Tipo según las etapas CBR.

Gestionar Usuarios

Esta ventana permite crear, modificar y borrar los distintos usuarios de la aplicación.

Acciones de usuario

Ejecutar ciclo CBR

Ejecutar un ciclo CBR (Razonamiento Basado en Casos) consiste en realizar una consulta para un Tipo de Caso concreto. Se pedirá al usuario los datos del problema y se le ofrecerá una solución, que podrá valorar de 0 a 100, hecho que se tendrá en cuenta en siguientes consultas.
La opción de ciclo CBR configurado está restringida a usuarios avanzados y permite, además, establecer la métrica (método de evaluación) de cada atributo, su peso a la hora de buscar una solución y las técnicas a emplear en cada fase del ciclo CBR.

Importar casos

La opción de Insertar Casos permite introducir casos para un determinado Tipo de Caso manualmente o desde un fichero WEKA, para que la Base de Casos de ese Tipo sea más completa

Estadisticas

Completas estadísticas del uso de la aplicación por parte de los usuarios. Los usuarios corrientes únicamente pueden ver sus propias estadísticas, mientras que los administradores pueden ver estadísticas globales o filtrar según usuarios y Tipos de Caso
