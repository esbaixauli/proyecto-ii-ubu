See case type



In this window you can see a previously chosen case type. You can also alter its attributes (Exclusively its weight and metric), as well as its methods. This way, an administrator can decide that a method must stop being used for that case, enable a method or change its default configuration. Adding or deleting attributes is not allowed.

This option's usage is very similar to that of Creating a new type. . Check Create a new case type to get more information.
