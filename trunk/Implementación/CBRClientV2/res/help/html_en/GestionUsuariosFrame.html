Users and their types



Users log into the application after being registered in it. The only users that can sign up new users are those of the Administrator type.

The user types are:
Administrators: They have access to all the system's features.
Advanced Users: They can execute both basic and configured CBR cycles, check their stats and insert cases of the types the Administrators have assigned to them.
Basic Users: They can perform the same actions of an advanced user, except for executing configured CBR cycles.

Actions over the users

The buttons at the lower area of the window allow to perform many operations over the users.
Visualize: It allows to see the selected user's data and modify them. This includes assigning/unassigning case types to an user.
Create: Allow to create a new user of any type, including administrators.
Delete: Deletes the selected user.
For security reasons, there always exists an administrator named 'root', unerasable.
