Crear un nuevo tipo de caso

Componentes de un tipo de caso

Un tipo de caso se compone de los siguientes elementos:
-Un conjunto de atributos del problema. Cada atributo posee un peso y una métrica de evaluación por defecto, que se emplearán en las consultas CBR si el usuario no introduce valores para estos campos.
-Un conjunto de atributos de la solución.
-Una serie de técnicas permitidas para cada etapa del ciclo. Entre ellas habrá una que será la empleada por defecto si el usuario ejecuta una consulta básica o no escoge una técnica en una consulta configurada (Ver apartado de la ayuda sobre consultas).
-La configuración de cada una de esas técnicas permitidas.

Añadir/borrar atributos



Para añadir un atributo, tanto al problema como a la solución se ha de pulsar el botón . Para borrarlo, pulse el botón . Nótese que debe existir al menos un atributo para el problema y uno para la solución.

Rellenar atributos

Es obligatorio rellenar todos los campos de cada atributos. Para ello, rellene el nombre de cada atributo (Que ha de ser único). Después ha de escoger el tipo del atributo. Los tipos permitidos son:
-Cadena de texto. P.ej:"Prueba"
-Número entero. P.ej:2
-Número decimal. P.ej:2,3
Finalmente, ha de escoger la métrica por defecto. Las métricas disponibles dependen del tipo de atributo. Por ejemplo, la comparación por subcadena más larga no está disponible para Numeros enteros o decimales. Algunas métricas requieren un parámetro que la configure (Ver apartado sobre conceptos básicos, sección Métricas)

Establecer técnicas



Para establecer las técnicas, pulse el botón "Configurar técnicas". Se comprobará la validez de los atributos introducidos y se mostrará el recuadro de técnicas, que posee un panel por cada etapa del ciclo CBR. Al lado de cada técnica existe una casilla de verificación que indica si esta técnica estará disponible para los usuarios (Siempre debe haber al menos una). En el desplegable de cada panel se permite escoger qué técnica se usará por defecto si el usuario no escoge ninguna cuando realiza una consulta.

Algunas técnicas se pueden configurar. Para ello, pulse en el botón "Configurar" al lado de cada técnica. Si escoge no configurar una técnica esta tomará valores por defecto.

Últimos pasos

Finalmente, pulse el botón Disquete para guardar el nuevo tipo de caso. Se comprobará:
-Que se hayan establecido las técnicas.
-Que se hayan rellenado los atributos y no contengan nombres repetidos.
-Que el nombre del tipo de caso no exista ya y no contenga espacios ni caracteres especiales.
