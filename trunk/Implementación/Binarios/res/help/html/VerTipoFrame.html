Ver tipo de caso



En esta pestaña se pueden ver el tipo de caso previamente escogido. También se pueden modificar sus atributos (Exclusivamente su peso y métrica), así como las técnicas. De este modo, el administrador puede decidir que una técnica deje de usarse para ese caso, habilitar una técnica o modificar su configuración por defecto. No se permite añadir o borrar atributos.

El uso de esta opción es muy similar al de Crear un nuevo tipo. Consúltese Crear un nuevo tipo de caso para más información.
